# 🏏 BPL - Cricket Dream 11 Team Selection App

A responsive cricket team selection web application built with React, Tailwind CSS, and DaisyUI. Users can select players, manage coins, and build their ultimate Dream 11 cricket squad.

---

## 🚀 Live Features

- ⚡ Responsive Design
- 🏏 Cricket Player Selection
- 💰 Virtual Coin System
- ✅ Add/Remove Selected Players
- 🔔 Toast Notifications
- 🎨 Modern UI with DaisyUI
- 📱 Mobile Friendly Layout
- 📬 Newsletter Section
- 🌙 Beautiful Footer Design

---

## 🛠️ Technologies Used

- React
- Tailwind CSS
- DaisyUI
- React Icons
- React Toastify
- Vite

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/bpl-cricket-app.git
```

Go to the project folder:

```bash
cd bpl-cricket-app
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

## 🎯 Core Functionalities

### 🪙 Coin Management
- Users start with virtual coins.
- Coins decrease when selecting players.

### 👥 Player Selection
- Add players to selected squad.
- Remove players anytime.

### 📱 Responsive UI
- Optimized for:
  - Mobile
  - Tablet
  - Desktop

### 🔔 Notifications
- Toast messages for:
  - Successful selection
  - Coin warnings
  - Duplicate player warnings
